# 흔들리는 집중 (Flickering Focus)

현대 지식인들이 직면한 가장 큰 문제 중 하나인 **'디지털 방해 요소 속에서의 집주력 상실'**을 풍자하고 유쾌하게 체험해 보는 4키 리듬 게임입니다.

본 프로젝트는 경인교육대학교(GINUE) AI창의컴퓨팅 기말과제를 위해 기획/구현되었으며, 별도의 서버나 데이터베이스 없이 브라우저 자체 오디오 신디사이징과 CSS 애니메이션을 이용해 동작하는 무설치 순수 웹 게임입니다.

---

## 🎮 게임 설명

* **조작키**: `D`, `F`, `J`, `K` (리듬 레인 대응)
* **목표**: 위에서 낙하하는 공부 노트(책 아이콘)들이 하단의 붉은 판정선에 도달할 때 알맞은 키를 입력하여 높은 집중도 점수를 획득합니다.
* **디지털 방해 요소**: 게임 진행 중 스마트폰 카카오톡 메시지, 디스코드 알림, 유튜브 라이브 알림, 악성 배너 광고 팝업 등이 무작위로 화면 전체에 생성되어 시야를 가리고 집주를 방해합니다.
* **해결법**: 팝업창이 뜨면 지체하지 말고 **마우스로 우측 상단의 `✕` 버튼을 클릭**해 닫아주어야 뒤에서 내려오는 노트를 볼 수 있습니다.
* **난이도 변화**: 시간이 지남에 따라(15초 간격) 노트의 낙하 속도가 빨라지며, 방해 팝업 창의 발생 빈도가 급증합니다.
* **게임 시간**: 총 60초 (1분 자동 종료)

---

## 🏆 집중도 평가 기준

게임이 종료되면 누적 점수(집중도)에 따라 3가지 평가 등급을 얻게 됩니다.
1. **300점 이상**: 🏆 **집중력 마스터 (Focus Master)**
   - 대단한 멀티태스킹과 엄청난 집중력의 소유자!
2. **150점 이상 300점 미만**: 🥈 **집중력 우수 (Excellent Focus)**
   - 방해 요소를 극복했지만, 가끔씩 주의가 흐트러진 일반적인 우수 상태.
3. **150점 미만**: ⚠️ **주의 분산 상태 (Distracted State)**
   - 스마트폰과 유튜브, 광고에 영혼을 잠식당해 공부를 전혀 하지 못한 현대인 평균 상태.

---

## 📁 프로젝트 파일 구성

* `index.html`: 게임 화면 레이아웃 및 마크업 구조
* `style.css`: 네온 스타일 다크 테마, 애니메이션 효과, 글래스모피즘 알림창 디자인
* `game.js`: 게임 엔진 루프, Web Audio API를 활용한 드럼/베이스/멜로디 실시간 사운드 합성기, 판정 및 방해 장치 로직
* `README.md`: 프로젝트 실행 가이드 및 설명서 (본 파일)

---

## 🚀 실행 및 깃허브 배포 방법

### 1. 로컬에서 바로 실행해보기
1. 폴더 내 `index.html` 파일을 크롬(Chrome), 엣지(Edge), 사파리(Safari) 등 웹 브라우저 창으로 드래그 앤 드롭하거나 더블 클릭합니다.
2. 화면 중앙의 **[집중하기 시작]** 버튼을 클릭하면 사운드가 로드되며 카운트다운 후 게임이 플레이됩니다.

### 2. 깃허브(GitHub Pages)에 배포하여 온라인 실행하기
1. 깃허브([github.com](https://github.com))에 로그인 후 우측 상단의 **`New`** 버튼을 클릭하여 새 저장소(Repository)를 생성합니다.
   * 저장소 이름 예시: `shaking-focus`
   * **반드시 Public(공개)으로 설정해야 무료 호스팅이 가능합니다.**
2. 생성된 저장소에 `index.html`, `style.css`, `game.js`, `README.md` 파일을 모두 업로드합니다.
3. 저장소 페이지 상단의 **`Settings`** (설정) 메뉴로 들어갑니다.
4. 왼쪽 네비게이션 바에서 **`Pages`** 메뉴를 클릭합니다.
5. **Build and deployment** 섹션 하위의 **`Branch`** 설정을 `None`에서 `main`(또는 `master`)으로 변경하고, 우측의 폴더 경로는 `/ (root)` 그대로 유지한 채 **`Save`**를 누릅니다.
6. 1~2분 후 페이지 상단에 **`Your site is live at https://<사용자ID>.github.io/shaking-focus/`** 형태의 무료 배포 주소가 활성화됩니다.
7. 해당 주소를 모바일 혹은 컴퓨터 브라우저 주소창에 복사하여 전 세계 어디서든 플레이하고 공유할 수 있습니다!
